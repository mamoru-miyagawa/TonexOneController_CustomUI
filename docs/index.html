Tonex One Controller — Custom UI Flasher

Web-based installer for the customized UI fork.

Target hardware: Waveshare 3.5B (ESP32-S3) + Tonex One

How to flash

  1. Use Google Chrome or Microsoft Edge on a desktop computer. (Firefox and Safari don't support the Web Serial API.)
  2. Press and hold the BOOT button on the Waveshare board.
  3. Connect a USB-C cable from the board to your computer.
  4. Release the BOOT button.
  5. Click the Connect button below and select the new serial port that appears (often labeled "USB JTAG/serial debug unit" or similar).
  6. Follow the on-screen prompts. When finished, unplug the cable and power the board normally to test it.

If the button stays disabled, your browser doesn't support Web Serial — switch to Chrome or Edge.

What's installed?

This is the customized UI fork of Builty/TonexOneController. The UI was rebuilt for a more compact arc-based layout on the 480×320 screen. See the repo README for a list of changes.

Troubleshooting

  • "No matching device" — make sure you held BOOT while plugging in. Unplug, hold BOOT, plug back in, release BOOT, retry.
  • Flash succeeds but screen stays blank — unplug USB and connect a normal power source. The screen only initializes on a fresh power-up after flashing.
  • Pedal not responding after flash — try again with the "Erase device" option enabled. Note this wipes saved settings on the controller.
